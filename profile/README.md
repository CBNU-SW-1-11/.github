## 👥 팀 소개

충북대학교 소프트웨어학과에서 진행하는 졸업 프로젝트 팀입니다.

## 🚀 주요 프로젝트

### AI of AI
**다중 LLM 기반 답변 최적화 시스템**

- 🤖 여러 대형 언어 모델(GPT, Gemini, Claude)을 통합하여 최적의 답변 생성
- ⚖️ Judge 모델을 통한 답변 검증 및 평가
- 🎬 YOLO, DeepFace, Ollama를 활용한 영상 분석
- 🌐 한국어 객체 검색 및 자연스러운 질의응답
- 🔍 Wikipedia, Wikidata 등 외부 소스를 활용한 사실 검증

## 🛠️ 기술 스택

### Backend
- Django & Django REST Framework
- YOLO (Ultralytics)
- DeepFace
- Ollama

### Frontend
- React
- Tailwind CSS

### AI/ML
- OpenAI API (GPT)
- Anthropic API (Claude)
- Google Gemini API
- YOLOv8
- LLaVA (Ollama)

## 📊 프로젝트 특징

### 1. 다중 LLM 통합
여러 AI 모델의 답변을 비교하여 가장 신뢰할 수 있는 답변을 선택합니다.

### 2. 영상 분석
- 객체 감지 (YOLO)
- 사람 속성 분석 (DeepFace)
- 자연어 캡션 생성 (Ollama)

### 3. 한국어 지원
한국어 질문으로 영상 내 객체를 검색하고 자연스러운 답변을 받을 수 있습니다.

### 4. 사실 검증
외부 지식 베이스를 활용하여 답변의 정확성을 검증합니다.

## 📚 주요 기능

- ✅ 다중 LLM 응답 수집 및 비교
- ✅ Judge 모델을 통한 최적 답변 선택
- ✅ 영상 업로드 및 자동 분석
- ✅ 한국어 객체 검색
- ✅ 색상/키워드/시간 기반 프레임 검색
- ✅ 사실 검증 시스템
- ✅ 세션별 대화 기록 관리

## 🌟 프로젝트 목표

1. **정확성 향상**: 여러 AI 모델의 답변을 비교하여 더 정확한 정보 제공
2. **사용자 경험**: 한국어로 자연스럽게 질의응답 가능
3. **영상 이해**: 영상 내용을 분석하고 질문에 답변
4. **사실 검증**: 외부 소스를 활용한 신뢰성 있는 정보 제공
